## Deskripsi
Repository ini berisi *starter code* untuk sesi praktikum **Pertemuan 3: Layouting dan Responsive CSS** . Dalam praktikum ini, kita akan belajar mengatur tata letak elemen pada halaman web (Layouting) dan membuatnya adaptif di berbagai ukuran layar (Responsive). 

Di akhir sesi, kita akan membangun sebuah halaman Portofolio Developer yang responsif dan siap digunakan!
